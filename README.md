# Giveaway-o-tron

Windows program to allow Twitch giveaways with multiple options.

## Development

```sh
git clone git@github.com:maael/giveaway-o-tron.git
cd giveaway-o-tron
npm i
neu update
npm run dev
```

## Build

```sh
npm run build
```

This will make a `dist/giveaway-o-tron` directory, that can be zipped and distributed.
